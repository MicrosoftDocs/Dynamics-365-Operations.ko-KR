# <a name="configure-rates"></a>요율 구성


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

요율은 고용주와 직원이 복리후생에 기여하는 금액을 정의합니다. 값은 구성에 따라 금액 또는 플렉스 크레딧 수일 수 있습니다.

요율을 사용하여 여러 요인에 따라 직원과 고용주가 각 복리후생에 대해 지불하는 금액을 결정합니다. 보장 요율은 날짜가 적용하므로 요율의 과거 기록을 유지할 수 있습니다. 

## <a name="set-up-rates"></a>요율 설정

1. **복리후생 관리** 작업 영역의 **설정** 에서 **요율** 을 선택합니다.

2. **새로 만들기** 를 선택합니다.

3. 다음 필드에 값을 지정합니다.

   | 필드 | 설명 |
   | --- | --- |
   | **요율** | 복리후생 요율을 식별하는 고유한 이름. |
   | **설명** | 복리후생 요율에 대한 설명. |
   | **유효** | 요율이 적용되는 날짜. 현재 시스템 날짜가 기본값입니다. 이 날짜는 복리후생 기간이거나 그 이전이어야 합니다. 이 날짜를 복리후생 계획 날짜로 설정하는 것이 좋습니다. |
   | **만료** | 요율의 종료 날짜. 기본값은 2154-12-31이며 종료되지 않음을 의미합니다. |
   | **계층 사용** |  요율을 결정하는 데 사용해야 하는 논리가 있는 경우 이 필드를 사용합니다. 예를 들어 나이에 따라 요율을 인상해야 하는 경우 여기에서 값을 선택합니다. 1단계 복리후생 요율에는 **단일 계층** 를 선택하고 2단계 복리후생 요율에는 **이중 계층** 을 선택합니다. 이중 계층의 예로는 성별 및 나이에 기반을 두는 계층이 있습니다. 값을 선택한 후 **작업** 을 선택한 다음 **계층 요율** 을 선택합니다. 변경되지 않는 고정 요율이 있는 경우 이 필드를 비워 둡니다. |
   | **지불 빈도** | 복리후생 공급자에게 복리후생 프리미엄 요율을 지불해야 하는 빈도를 지정합니다. 이 항목의 뒷부분에서 설명하는 페이지에 입력하는 요율은 여기에서 지정하는 지불 빈도에 기반을 둡니다. 예를 들어 이 필드에 **매월** 을 입력하고 직원 요율에 **$100** 를 입력하면 직원에게 매월 $100의 비용이 든다고 가정합니다. 하지만 직원 레코드에 설정된 복리후생 지불 빈도에 따라 월 2회 지불할 수 있습니다. 이 경우 직원이 **직원 셀프 서비스** 에 로그인하면 **직원 셀프 서비스** 가 표시하는 요율이 직원의 지불 빈도에 기반을 두므로 지불하는 금액은 $50입니다. |
   | **지불 빈도 요율 반올림** | 요율을 반올림하는 방법은 표준, 잘림, 정규, 하향, 반올림입니다. </br></br><ul><li>**표준** – 항상 반올림합니다. 예를 들어 10.611은 10.62로 반올림합니다. -10.231은 -10.23로 반올림합니다. </li><li>**잘림** – 항상 반내림합니다. 예를 들어 10.619는 10.61로 반올림합니다. -10.231은 -10.24로 반올림합니다. </li><li>**정규** – 5 이상으로 끝나는 10진수 값은 0에서 반올림됩니다. 4 이하로 끝나는 10진수 값은 0으로 반올림됩니다. 예를 들어 10.615는 10.62로 반올림합니다. -10.235는 -10.24로 반올림합니다. 10.614는 10.61로 반올림합니다. -10.234는 -10.23으로 반올림합니다. </li><li>**하향** – 0을 향해 반올림합니다. 예를 들어 10.619는 10.61로 반올림합니다. -10.231은 -10.23로 반올림합니다. </li><li>**반올림** – 0에서 반올림합니다. 예를 들어 10.619는 10.62로 반올림합니다. -10.231은 -10.24로 반올림합니다. |
   | **비흡연 직원 금액** | 복리후생 공급자가 비흡연 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **비흡연 고용주 금액** | 복리후생 공급자가 비흡연 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **흡연자 직원 금액** | 복리후생 공급자가 흡연하는 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **흡연자 고용주 금액** | 복리후생 공급자가 흡연하는 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **관리 금액** | 타사 관리자가 청구하는 관리 금액. 이는 고용주가 타사 관리자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **플렉스 크레딧 요율** | 복리후생에 드는 플렉스 크레딧 수입니다. 이는 플렉스 크레딧 프로그램과 연결된 복리후생 계획에 대한 요율에만 적용됩니다. 계층 요율을 사용하는 경우 플렉스 크레딧 요율은 작업 > 계층 요율에서 정의됩니다. |
   | **적용 날짜 변경** | 복리후생 요율 변경이 적용되는 날짜입니다. 요율 변경 업데이트 처리를 실행하면 시스템은 자동으로 복리후생 요율을 변경하고 이 요율과 관련된 모든 복리후생 계획을 업데이트합니다. 시스템이 이 요율을 기반으로 직원 복리후생 계획을 자동으로 업데이트하도록 하지 않으려면 이 날짜를 설정하지 마세요. 이는 일반적으로 향후 자동 요율 변경 처리를 위해 예약되어 있습니다. **변경 적용 날짜** 는 복리후생 요율 적용 및 만료 날짜 이내여야 합니다. |
   | **요율 변경 완료** | 요율 변경 업데이트 처리를 통해 복리후생 요율 변경이 완료되면 **요율 변경 완료** 확인란이 자동으로 선택됩니다. |

4. 복리후생 요율 설정에 대한 변경 사항을 추적하고 유지하려면 **작업** 을 선택한 다음 **버전 유지** 를 선택합니다.

5. **저장** 을 선택합니다. 

## <a name="set-up-tier-rates"></a>계층 요율 설정

요율이 다양한 요인에 따라 달라지는 경우 요율 설정에서 계층 요율을 사용할 수 있습니다. 예를 들어, 나이가 최대 34.99세인 비흡연자는 금액이 2가 되도록 계층 요율을 구성할 수 있습니다. 나이가 최대 39.99세인 비흡연자는 금액이 3입니다.

이중 계층을 사용할 수도 있습니다. **요율 설정** 양식의 **계층 사용** 값으로 **이중 계층** 을 선택하면 두 차원을 기반으로 요율을 정의할 수 있습니다. 예를 들어, 나이가 최대 34.99세인 남자 비흡연자는 금액이 2가 되도록 이중 계층 시스템을 구성할 수 있습니다. 나이가 최대 39.99세인 남자 비흡연자는 금액이 3입니다. 나이가 최대 34.99세인 여자 비흡연자는 금액이 1.8입니다. 나이가 최대 39.99세인 여자 비흡연자는 금액이 2.8입니다.

> [!IMPORTANT]
> 작업자 기록의 **개인 정보** 에는 직원이 흡연자인지 지정하는 옵션이 있습니다. 직원이 흡연자로 기록된 경우 흡연자 요율이 사용됩니다. (흡연자 표시는 직원에게 표시되지 않습니다.)
   
1. **복리후생 관리** 작업 영역의 **설정** 에서 **요율** 을 선택합니다.

2. 목록에서 하나 이상의 요율을 선택하고 **작업** 을 선택한 다음 **계층 요율** 을 선택합니다.

3. **새로 만들기** 를 선택합니다.

3. 다음 필드에 값을 지정합니다.

   | 필드 | 설명 |
   | --- | --- | 
   | **설명** | **설명** 필드 값은 요율 설정 레코드의 설명에서 적용됩니다. 이는 계층 요율이 연결된 요율 설정을 식별하는 데 도움이 됩니다. |
   | **계층 코드** | 계층 코드를 선택합니다. 계층 코드는 **계층 코드** 페이지에 정의됩니다. 시스템은 왼쪽의 그리드에 자동으로 계층 코드에 관한 설명을 표시합니다. |
   | **계층 유형** | 계층 요율 계산 프로세스의 선택 기준으로 사용할 필드를 지정합니다. 예:</br></br><ul><li>**나이** 를 사용하는 경우 시스템은 복리후생 요율 계산 프로세스에 직원의 생년월일을 사용합니다.</li><li>**급여** 를 사용하는 경우 시스템은 복리후생 요율 계산 프로세스에 직원의 연간 복리후생 급여를 사용합니다.</li><li>**직무 유형** 을 사용하는 경우 직원의 현재 활성 직위 레코드를 사용하여 직위와 연결된 직무 레코드로 직무 유형을 결정합니다.</li></ul></br></br>계층 유형은 **나이**, **급여**, **신체**, **성별**, **정규직 상응**, **직무 유형**, **보상 지역**, **수준** 이 있습니다. | 
   | **수준** | 복리후생 요율 계산 프로세스 중에 계층 유형과 함께 사용할 값. 예:</br></br><ul><li>계층 유형이 **나이** 이면 나이가 사용됩니다.</li><li>계층 유형이 **급여** 이면 급여 금액이 사용됩니다.</li><li> 계층 유형이 **직무 유형** 이면 직무 유형이 사용됩니다.</li></ul></br></br>계층 유형이 **나이** 또는 **급여** 인 경우 **수준** 필드의 값은 계층의 상한을 나타냅니다. **직무 유형** 의 계층 유형에서는 계층 요율 선택 시 정확히 일치 접근 방식이 사용됩니다. |
   | **계산 유형** | 계산 금액 필드의 금액을 사용하는 방법과 필요한 경우 수행할 산술 계산을 지정합니다. 계산 유형이 고정 금액인 경우 금액 필드가 그대로 사용됩니다. 계산 유형이 $ 금액의 급여 또는 보장 기준인 경우 계산 금액이 사용되며 계산 방향은 산술 계산에 사용됩니다.</br></br>계산 유형이 $ 급여 금액 기준인 경우 다음 산술 방정식이 사용됩니다.</br></br>직원 또는 고용주의 흡연자 또는 비흡연자 금액을 계산 금액(반올림 또는 반내림)으로 나눈 연간 복리후생 급여.</br></br>계산 유형이 $ 보장 금액 기준인 경우 다음 산술 방정식이 사용됩니다.</br></br>직원 또는 고용주의 흡연자 또는 비흡연자 금액을 계산 금액(반올림 또는 반내림)으로 나눈 보장 금액.</br></br>두 계산 모두에서 계산 방향을 사용하여 연간 복리후생 급여 또는 보증 금액을 계산 금액으로 반올림할지 또는 반내림할지 결정합니다. |
   | **계산 금액** | 복리후생 요율 계산 프로세스 중에 사용할 금액. 이 금액은 계층 요율을 산술적으로 계산할 때 제수가 됩니다. |
   | **계산 방향** | 계산된 결과 금액을 반올림하는 방향. 시스템은 공백(정확한 방법), **증가** 및 **감소** 의 세 가지 계산 방향을 지원합니다.</br></br><ul><li>공백인 경우 시스템은 급여/보장 금액을 계산 금액으로 나눈 정확한 계산을 사용합니다. 이 값에 분수가 있으면 계산에 사용됩니다.</li><li>**증가** 인 경우 산술 계산으로 급여/보장 금액을 계산 금액으로 나누고 정수로 높입니다. 즉, 12.25가 13으로 증가합니다.</li><li>**감소** 인 경우 산술 계산으로 급여/보장 금액을 계산 금액으로 나누고 현재 정수로 낮춥니다. 즉, 12.25가 12로 감소합니다.</li></ul> |
   | **비흡연 직원 금액** | 복리후생 공급자가 비흡연 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **비흡연 고용주 금액** | 복리후생 공급자가 비흡연 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
   | **흡연자 직원 금액** | 복리후생 공급자가 비흡연 직원에게 청구하는 금액. 이는 고용주가 복리후생 공급자에게 지불하는 금액이며 요율 설정에 대한 지불 빈도를 기반으로 해야 합니다. |
