# <a name="define-new-jobs"></a>새 직무 정의


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



이 작업은 새 직무 만들기에 중점을 둡니다. 템플릿 기능에서 수동 입력과 복사를 사용하여 완성됩니다. 이 작업을 만드는 데 사용된 데모 데이터 회사는 USMF입니다. 이 작업은 인적 자원 전문가를 대상으로 합니다.

1. **Human Resources \> 직무 \> 직무** 로 이동합니다.
2. **새로 만들기** 를 선택합니다.
3. **직무** 필드에 값을 입력합니다.
4. **설명** 필드에 값을 입력합니다.
5. 직무에 사용할 표준 직함을 선택합니다.

    이 직함은 직무에 할당된 새 직위의 기본값으로 사용됩니다.

6. 목록에서 원하는 레코드를 찾아 선택합니다.
7. 목록에서 선택한 행의 링크를 선택합니다.
8. **최대 직위 수** 그룹에서 **최대 직위** 를 선택합니다.
9. 최대 직위 수를 입력합니다. 

    이 수를 초과하는 직위 할당은 허용되지 않습니다.

10. **설명** 섹션의 **참고** 필드에 **직무에 대한 표준 설명 입력** 을 입력합니다.

    이 설명은 정의하는 직무에 할당된 모든 직위에 적용됩니다.

11. **직무 분류** 섹션에서 Human Resources의 표준 보고에 사용할 직무 기능을 선택합니다.
12. 목록에서 원하는 레코드를 찾아 선택합니다.
13. 목록에서 선택한 행의 링크를 선택합니다.
14. **직무 유형** 필드에서 조직의 직무를 추가로 분류할 직무 유형을 선택합니다.

    작업 유형은 Human Resources 전반에 걸쳐 보고하는 데 사용할 수 있습니다.

15. 목록에서 원하는 레코드를 찾아 선택합니다.
16. 목록에서 선택한 행의 링크를 선택합니다.
17. **보상** 섹션에서 보상 수준을 선택합니다.

    보상 수준은 직원이 해당 직무에 대해 받을 수 있는 최소 및 최대 보상을 정의합니다. 보상 규칙은 직원 보상이 범위를 벗어날 수 있는지를 결정합니다.

18. 목록에서 원하는 레코드를 찾아 선택합니다.
19. 목록에서 선택한 행의 링크를 선택합니다.
20. **기술** 섹션에서 **기술**, **교육**, **자격증** 빠른 탭은 비어 있습니다. 직무 템플릿에서 정보를 복사한 후 한 단계에서 이러한 필드에 정보를 입력합니다.
21. **저장** 을 선택합니다.
22. **다음에서 복사** 를 선택합니다.
23. **템플릿에서 복사** 를 선택합니다.
24. **직무 템플릿** 필드에서 드롭다운 화살표를 선택하여 조회를 엽니다.
25. 목록에서 원하는 레코드를 찾아 선택합니다.
26. 목록에서 선택한 행의 링크를 선택합니다.
27. **메모** 옵션을 선택합니다.

    선택한 항목만 선택한 직무에 복사됩니다.

28. **확인** 을 선택합니다.
29. **자격증** 섹션을 확장합니다.
30. **교육** 섹션을 확장합니다.
31. **저장** 을 선택합니다.
