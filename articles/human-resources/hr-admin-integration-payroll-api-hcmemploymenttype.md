# <a name="payroll---employment-type-option-set"></a>급여 - 고용 유형 옵션 집합

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

이 항목에서는 Dynamics 365 Human Resources의 고용 유형 옵션 집합에 관해 설명합니다.

## <a name="option-set-employment-type"></a>옵션 집합: 고용 유형

이 열거형은 직원 고용 유형 값의 옵션 집합을 제공합니다. 이것은 mshr_hcmemploymenttype 옵션 집합에서 사용할 수 있습니다.

| 값 | 레이블 | 설명 |
| --- | --- | --- |
| 200000000 | 직원 | 직원 |
| 200000001 | 하청업체 | 하청업체 |
