# <a name="identify-and-deploy-candidate-selection-tools"></a>후보 선택 도구 식별 및 배포

[!include [banner](../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

공석을 채울 적격한 후보자 풀을 찾는 것은 어려울 수 있습니다. 특히 직책에 고유한 기술 집합이 필요한 경우에는 더욱 그렇습니다. 그러나 필요한 기술을 갖춘 후보자가 이미 조직에 고용되어 있을 수 있습니다. 기존 직원 또는 신규 지원자 중에서 특정 스킬 세트를 검색할 수 있습니다. 이를 통해 채용 담당자는 현재 또는 과거에 공석에 지원한 지원자를 신속하게 수집 및 선별하거나 기존 직원 풀에서 잠재적인 후보자를 찾을 수 있습니다. 이 절차를 사용하여 기술 매핑 기능이 공석에 적합한 사람을 찾는 데 어떻게 도움이 되는지 알아보세요. **USMF** 데모 데이터 회사가 이 절차를 만드는 데 사용되었습니다.

1. **인저 자원 \> 역량 \> 기술 분석 \> 기술 매핑 프로필** 로 이동합니다.
2. **새로 만들기** 를 선택합니다.
3. **기술 매핑** 필드에서 기술 매핑의 이름을 입력합니다. 예를 들어 **회계사** 를 입력합니다.
4. **설명** 필드에 기술 매핑에 대한 설명을 입력합니다.
5. **날짜** 필드에 날짜를 입력합니다.
6. **프로필 검색** 을 선택하여 선택한 사람, 직업 또는 과정에서 인증서, 기술 및 교육 데이터를 가져와 검색의 기초로 사용할 수 있습니다. 그런 다음 기준을 추가 또는 제거하고 선택 사항인지 여부를 표시하고 중요도의 순위를 지정할 수 있습니다.
7. **작업** 을 선택합니다.
8. **작업** 필드에 값을 입력하거나 선택합니다.
9. **확인** 을 선택합니다.
10. **범위** 빠른 탭에서 부서와 같은 추가 정보를 추가합니다.
11. **인증서** 빠른 탭에서 인증서를 보거나 편집합니다.
12. **기술** 빠른 탭에서 기술을 보거나 편집합니다.
13. **교육** 빠른 탭에서 교육 기준을 보거나 편집합니다.
14. **실행** 을 선택합니다.
15. **확인** 을 선택합니다.
16. **결과** 를 선택합니다.
17. **결과** 를 선택합니다.
18. **계속하기** 를 선택합니다.
19. **인증서** 를 선택합니다.

    목록에 있는 각 사람을 자세히 분석하여 교육, 기술 및 전문적 경험에 대한 세부 정보를 볼 수 있습니다.

20. 페이지를 닫습니다.
21. 페이지를 닫습니다.
22. 결과를 다시 선택합니다.
23. **보고서** 를 선택합니다.

    가장 일치하는 항목이 보고서의 맨 위에 나열됩니다. 간격 요소가 나열되어 있어야 합니다. 이 요소는 스킬 매핑에 나열된 레벨과 개인에게 할당된 스킬 레벨 간의 차이를 나타냅니다.

24. 페이지를 닫습니다.
25. **저장** 을 선택합니다.
